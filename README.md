# Métodos Numéricos en Física

Repositorio con notebooks de Python de un curso de pregrado en análisis numérico aplicado a física. En desarrollo.

## Descripción
Notebooks que cubren métodos como Euler y Runge-Kutta para simulaciones físicas. Open-source, úsalos y mejora.

## Instrucciones
- Instala Jupyter Notebook.
- Requiere: NumPy, Matplotlib.
- Ejecuta los notebooks en orden sugerido.
